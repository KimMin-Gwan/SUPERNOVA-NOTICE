# 📢 SUPERNOVA 업데이트 안내 ( 26.02.10 / V2.3.0 )

안녕하세요.
**TEAM SUPERNOVA 대표 김민관**입니다.

이번 V2.3.0 업데이트는 스트리머 분들의 사용 편의성을 개선하고, 필요한 기능들을 추가하는 데 중점을 두었습니다.

<br/>
---

## ⚙️ 업데이트 일정

- **일시**: 2026년 02월 10일 (화) ~ 2026년 02월 13일 (금)
- **참고 사항**: 신규 기능 적용을 위한 업데이트 기간 중 서비스가 일시적으로 불안정할 수 있습니다. 이용 중 문제가 발생할 경우 제보해 주시면 신속히 조치하겠습니다.

<br/>
---

## 1️⃣ '일정표 추출하기' 기능 복구

- **기능 소개**
  - 작성한 일정표를 외부 커뮤니티나 SNS에 공유하기 편리하도록 **이미지 파일(PNG 또는 WEBP)로 저장**하는 기능이 돌아왔습니다.

- **상세 내용**
  - 추출하기 기능은 [스트리머 모드]에서 사용 가능합니다.
  - 원하는 날짜 범위와 레이아웃, 배경 이미지를 선택하여 저장할 수 있습니다.
  - *참고: 추출된 이미지는 최적화를 위해 웹상의 주간 일정표 레이아웃과 일부 디테일이 다를 수 있습니다.*

<br/>
---

## 2️⃣ 스트리머 페이지 개선

- **상단 배너 설정 기능 추가**
  - 스트리머 페이지 최상단에 **5:1 비율의 배너**를 설정할 수 있는 기능이 추가되었습니다.
  - 이미지를 별도로 설정하지 않을 경우, 프로필 이미지를 기반으로 자동 생성됩니다.
  - 배너 설정은 [스트리머 모드]에서 변경 가능합니다.

- **바로가기 숏컷 편집 기능 추가**
  - 기존 플랫폼 아이콘만 제공되던 방식에서, 이제 **원하는 링크와 이미지로 숏컷을 커스텀**할 수 있습니다.
  - 방송 플랫폼 링크가 최우선으로 노출되며, 그 뒤로 이어지는 숏컷 순서와 구성을 자유롭게 편집할 수 있습니다.
  - 숏컷 편집은 [스트리머 모드]에서 설정 가능합니다.

<br/>
---

## 3️⃣ 신규 주간 일정표 레이아웃 3종 추가

사용자분들께 큰 사랑을 받은 '노트' 레이아웃이 시리즈로 확장되었습니다.

- **노트 - 자연 (Nature)**: 녹색을 기반으로 한 차분하고 편안한 느낌의 테마
- **노트 - 고양이 (Cat)**: 황색 기반의 따뜻하고 귀여운 느낌의 테마
- **노트 - 게임 (Game)**: 보라색 기반의 활기차고 게이밍 감성의 테마

<br/>
---

## 4️⃣ 기타 업데이트 및 개선 사항

- **네이버 소셜 로그인 도입**
  - 네이버 공식 API를 연동한 소셜 로그인 기능이 추가되었습니다.
  - *기존 계정과의 연동 및 마이그레이션 기능은 추후 업데이트될 예정입니다.*

- **홈페이지 광고 배너 영역 추가**
  - 홈페이지 최상단에 제휴사를 위한 광고 배너 영역이 신설되었습니다.

- **슈퍼노바 퀘스트(설명 페이지) 오픈**
  - 추후 공개될 광고(슈퍼노바 퀘스트) 시스템의 소개 페이지가 추가되었습니다.
  - 게임사 관계자분들은 해당 페이지에서 이메일을 통해 사전 등록이 가능합니다.

- **기능 개선 및 버그 수정**
  - **월간 일정표**: 특정 일정이 잘못 표시되던 오류를 수정했습니다.
  - **일정 탐색**: 이제 탐색 페이지 리스트에서도 태그 정보를 한눈에 확인할 수 있습니다.
  - **태그 작성**: 일정 등록 시 태그 입력 경험(UX)을 더 편리하게 개선했습니다.

- **보안 강화**: 최근 감지된 다수의 서버 취약점 공격 시도에 대응하여, 서버 보안 시스템을 강화했습니다. 안심하고 이용하시기 바랍니다.

이번 업데이트는 스트리머 사용자분들의 **편의성을 개선**하고, 더 효과적이고 좋은 기능들로 스트리머 페이지를 채우고자 준비했습니다.
특히, 일정표 레이아웃이 많이 부족하다고 느꼈고, 앞으로 '노트' 시리즈처럼 다채로운 일정표 레이아웃을 주기적으로 선보일 예정입니다.

[슈퍼노바 퀘스트]는 TEAM SUPERNOVA가 야심 차게 준비한 **수익화 구조**입니다.
게임사와 스트리머를 연결하는 **마케팅 중개 플랫폼** 형태이며, 자세한 내용은 추후 추가로 공개될 예정입니다.

다가오는 설 명절, 시청자분들은 최애와 함께, 스트리머분들은 팬들과 함께하는, 따뜻하고 즐거운 연휴 되시길 바랍니다.
모두 새해 복 많이 받으시길 바라며, 저희는 더 좋은 기능으로 찾아뵙겠습니다.
