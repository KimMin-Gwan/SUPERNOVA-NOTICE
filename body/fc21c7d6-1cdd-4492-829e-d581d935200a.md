# 📢 SUPERNOVA 업데이트 안내 ( 26.03.11 / V2.4.0 )

안녕하세요.
**TEAM SUPERNOVA 대표 김민관**입니다.

이번 V2.4.0 업데이트는 사용자분들이 기다리시던 '일정표 커스텀' 기능을 포함하여 다양한 사용성 개선 작업이 추가되었습니다.

<br/>

---

## ⚙️ 업데이트 일정

- **일시**: 2026년 03월 11일 (수) ~ 2026년 03월 13일 (금)
- **참고 사항**: 신규 기능 적용을 위한 업데이트 기간 중 서비스가 일시적으로 불안정할 수 있습니다. 이용 중 문제가 발생할 경우 제보해 주시면 신속히 조치하겠습니다.

<br/>

---

## 1️⃣ '일정표 커스텀' 기능 추가

- **기능 소개**
  - 주간 일정표를 직접 꾸미고 만들어 볼 수 있는 커스텀 에디터 기능이 추가되었습니다.

- **커스텀 에디터 관련 기능**
  - 스티커 꾸미기 툴
  - 레이아웃 선택 기능
  - 일정표 테두리 및 속지 선택 기능
  - 제목 및 소제목 디자인 기능
  - 샘플 일정 추가 기능
  - 일정 카드 디자인 기능
  - 사진 & 마스킹 테이프 디자인 기능
  - 모바일 레이아웃 커스텀 기능

- **스티커 사용 관련**
  - 기본적으로 제공되는 아이콘 스티커 외에도, 직접 이미지를 업로드하여 커스텀 스티커로 사용할 수 있습니다.
  - (단, 업로드된 이미지 중 이용약관에 위배되는 이미지는 예고 없이 삭제될 수 있습니다.)

- **추출 전용 에디터 지원**
  - 커스텀으로 제작한 주간 일정표를 이미지로 깔끔하게 저장할 수 있도록 추출 전용 에디터가 추가되었습니다.
  - 기존 일정표는 웹 사이트 환경에 맞춰 구성되어 있어 이미지 추출에 어려움이 있었습니다. 이를 해결하고 원하는 레이아웃으로 변경하여 다운로드할 수 있도록 마련된 전용 에디터입니다.

<br/>

---

## 2️⃣ 일정 작성 기능 개선

- **일정 썸네일 크롭 기능 추가**
  - 일정에 포함되는 썸네일을 다양한 비율로 편집하여 업로드하는 기능이 추가되었습니다.
  - 지원 비율: [16:9], [9:16], [3:4], [4:3], [1:1] 및 원본 이미지

- **월간 달력 미리보기 기능 추가**
  - 일정 작성 탭에서 주간 일정표뿐만 아니라 월간 일정표 형태로도 확인할 수 있도록 개선되었습니다.

- **모바일 버전 미리보기 기능 추가**
  - 모바일 환경의 시청자들에게 일정표가 어떻게 표시되는지 미리 확인할 수 있는 기능이 추가되었습니다.

- **레이아웃 선택창 UI 변경**
  - 주간 레이아웃 선택 창에 '커스텀 일정표' 옵션이 추가되었습니다.
  - 기본 레이아웃은 '그리드 모드'와 '리스트 모드'를 각각 명확하게 선택할 수 있도록 변경되었습니다.

<br/>

---

## 3️⃣ 기타 업데이트 및 개선 사항

- **스트리머 페이지의 달력 UI 변경**
  - 달력의 크기를 더 키워 일정을 한눈에 보기 편하도록 가독성을 높였습니다.
  - 모바일 환경에서는 달력 내부에 일정이 표시되던 방식에서, 달력 하단에 일정이 목록 형태로 표시되도록 변경되었습니다.

- **클라이언트 관련 기능 서버 업데이트**
  - 추후 선보일 클라이언트 전용 기능들을 위한 서버 사전 업데이트가 진행되었습니다.

이번 업데이트의 핵심은 주간 일정표를 직접 제작할 수 있는 **'커스텀 일정표'** 기능입니다.
그동안은 운영진이 제공하는 레이아웃 안에서만 일정표를 작성해야 했지만, 스트리머분들이 더 다채롭고 자신만의 색깔이 담긴 일정표를 제작하여 시청자 및 팬분들과 소통하는 것이 옳다고 판단하여 준비하게 되었습니다.

물론, 커스텀 기능이 추가되었다고 해서 기존 레이아웃 제공이 중단되는 것은 아닙니다. 직접 꾸미는 것이 번거로우신 분들을 위해 기존의 깔끔한 기본 레이아웃들도 꾸준히 지원하고 업데이트할 예정입니다.

이번 V2.4.0은 규모가 상당히 큰 패치인 만큼, 지속적으로 최적화 작업과 버그 수정을 진행하고 있습니다. 이용 중 불편하신 점이 있다면 언제든 편하게 제보해 주시기 바랍니다.

아울러, 이어지는 **2.4.X 버전**들에서는 다음 기능들이 추가될 예정입니다.
- 네이버 계정 연동 (마이그레이션) 기능
- 네이버 카페 공지글 자동 게시 기능
- 비즈니스 전용 기능 및 광고 기능
- 모바일 앱 출시 및 전용 기능 추가

저희 팀 슈퍼노바는 앞으로도 사용자 여러분의 목소리에 귀 기울이며 꼭 필요한 기능들을 제공하기 위해 최선을 다하겠습니다.
