## 1️⃣ 사용성 개선

- **하루에 여러 방송 일정 작성 가능**  
  하루 2개 이상의 방송 일정(1부, 2부 등)을 등록할 수 있으며,  
  조회 시에도 각 일정을 개별적으로 확인할 수 있습니다.

   - 기존 : 하루에 방송 일정 한개만 생성 가능
   - 변경 : 하루에 여러개의 방송 일정 생성 가능

- **방송 일정 종류 선택 가능**  
  방송 일정 작성 시, 일정의 종류를 선택할 수 있습니다:  

  - 선택 가능한 일정 목록
    - 일반 방송  
    - 휴방  
    - 게릴라  
    - 공지/메모  

- **방송 플랫폼 선택 기능**  
  방송 일정 작성 시, 플랫폼을 선택할 수 있습니다:  
  - 일정의 종류가 일반 방송일 때만 선택 가능
  - 선택 가능한 플랫폼: **치지직, SOOP, YouTube**  
  - 사전에 **계정 옵션에서 플랫폼 정보 입력** 필요

- **날짜별 방송 일정 탐색**  
  원하는 날짜의 방송 일정을 쉽게 확인 가능  
  - **팔로우 중인 스트리머 일정만 보기** 기능 추가

- **팔로우 스트리머 확인 용이**  
  팔로우 중인 스트리머를 빠르게 확인할 수 있도록 UI/UX 개선

- **일정 달력 기능 정식 추가**  
  달력을 통해 일정을 한눈에 확인하고, 작성 및 관리가 편리하도록 개선

- **스트리머의 이름을 12글자로 제한**  
  너무 긴 스트리머 이름은 화면에서 잘려 보이기 때문에 
  9글자로 제한 했던 스트리머 이름 길이를 12글자까지 제한으로 변경
  - 기존 : 최대 9글자
  - 변경 : 최대 12글자 (공백 포함)

## 2️⃣ 서비스 기능 추가

- **스트리머와 일반 사용자 구분**  
  - 기존에는 모든 사용자가 동일하게 서비스를 이용했으나, 이제 **스트리머 모드**를 통해 기능을 구분
  - **스트리머 모드 기능**:  
    - 스트리머 정보 관리  
    - 방송 일정 작성  
    - 상단 네비게이션 바 또는 더보기 페이지에서 모드 전환 가능
  - **일반 사용자 모드 기능**:  
    - 스트리머 팔로우
    - 일정 구독
    - 팔로우 중인 스트리머 일정 모아보기
    - 구독중인 일정 모아보기
    - 날짜 별 일정 탐색
    - 태그 별 일정 탐색
    - 일정 달력 기능 (팔로우 중인 스트리머만 확인 가능)
    - 후기 작성 및 후기 게시판 탐색
    - 일정 및 후기 검색


- **방송 플랫폼 URL 등록**  
  - 스트리머 모드에서 활동 플랫폼 URL 작성 가능  
  - URL 등록 시, 해당 플랫폼 표시  
  - URL 미등록 시, 플랫폼 표시되지 않음  
  - 등록된 플랫폼은 방송 일정 작성 시 선택 가능  
  - 사용 가능한 플랫폼: **치지직, SOOP, YouTube**  
    - YouTube 단독 사용 시 커뮤니티 지침에 따라 불이익 발생
