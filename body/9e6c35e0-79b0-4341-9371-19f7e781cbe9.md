## 2️⃣ 포스트잇 접근성 향상

* **스트리머와 시청자 간의 원활한 소통을 위해 기능이 개선되었습니다.**
    * **스트리머 페이지 내 작성**: 이제 스트리머 페이지에서 바로 포스트잇 작성이 가능합니다.
    * **자동 공개 범위 설정**:
        * 스트리머 페이지에서 작성 시: 해당 스트리머만 볼 수 있도록 자동 설정됩니다.
        * 스트리머가 직접 작성 시: 팔로워만 볼 수 있도록 자동 설정됩니다.
